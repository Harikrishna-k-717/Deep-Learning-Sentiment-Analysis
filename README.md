# Deep Learning Sentiment Analysis Engine
## Project Overview
This project uses a **Long Short-Term Memory (LSTM)** neural network to analyze and classify the sentiment of movie reviews. 

### Key Technical Achievements:
* **Model Accuracy:** 85.01% on the IMDB dataset.
* **Architecture:** 128-dim Embedding layer followed by a 64-unit LSTM.
* **Data Handling:** Implemented padding and tokenization for 10,000 unique words.

### Analyst Insights:
* **Business Impact:** This model can be used by streaming platforms to automatically categorize viewer feedback.
* **Observation:** The model shows high sensitivity to superlative words ("amazing", "terrible"), proving effective for strong emotional feedback.
